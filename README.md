
# TREE

TREE (Three-dimensional Rendered Ensemble Environment) is a scientific visualization package (in early development) that utilizes Blender's rendering engine and 
Python API to produce reproducible, publication-quality visualizations. It uses a C++ core for high performance calculations that is tied to Blender through
Python bindings. It is lightweight and extends Blender's intuitive user interface. TREE is highly extensible and can be used by both developers
looking to fine tune their visualizations, and as an end product for those who need visualizations quickly.
